# Resume
Backend resources created for the Cloud Resume Challenge using Terraform with AWS as the cloud provider.
Resources include:
  1. API Gateway
  2. Cloudfront
  3. Dynamodb
  4. Lambda Function
  5. Route53
  6. S3 for a static website
  7. S3 for tfstate file storage
  8. ACM Certificate
  9. IAM Policies
