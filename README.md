# Resume
Backend resources created for the Cloud Resume Challenge using Terraform with AWS as the cloud provider.
Resources include:
  1. Apigateway
  2. Cloudfront
  3. Dynamodb
  4. Lambda Function
  5. Route53
  6. S3 for a static website with CORS configuration
  7. S3 for tfstate file
  8. ACM Certificates
  9. IAM Policies
